## 4. 仍需注意的“非硬约束”点

1. Eq.(12)/(14) 在工程实现中为软约束（`slack`+大惩罚），当资源不足时允许少量超期，避免模型不可行直接中断。  
2. `P_tilde/Q_tilde` 由于原始数据缺少空驶接客显式标签，使用需求强度校准近似，不是直接观测统计。  
3. 车队 MPC 使用 `transition_topk` 做稀疏化控制规模，属于可计算性近似。
